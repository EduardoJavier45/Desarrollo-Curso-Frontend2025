3. ARIA (Accessible Rich Internet Applications)

Acordeón Accesible

ARIA (Accessible Rich Internet Applications) es un conjunto de atributos que hacen el contenido web y las aplicaciones web más accesibles para personas con discapacidades.

Primera regla de ARIA: No usar ARIA si existe un elemento HTML nativo que proporciona la semántica necesaria. Usa ARIA solo cuando sea necesario para hacer contenido dinámico accesible.

  • aria-label: Proporciona etiqueta accesible
  • aria-labelledby: Referencia a elemento etiqueta
  • aria-describedby: Proporciona descripción adicional
  • aria-live: Anuncia cambios dinámicos
  • aria-expanded: Estado expandido/colapsado

5. Contraste de Colores

El contraste de color es crucial para usuarios con baja visión o daltonismo.

Ratios WCAG

  • Nivel AA: 4.5:1 para texto normal, 3:1 para texto grande
  • Nivel AAA: 7:1 para texto normal, 4.5:1 para texto grande
