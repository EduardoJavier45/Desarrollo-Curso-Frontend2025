📦 Modelo de Caja CSS

Entendiendo cómo CSS calcula el espacio de cada elemento

¿Qué es el Modelo de Caja?

En CSS, cada elemento es una "caja" rectangular compuesta por 4 áreas:

MARGIN (margen externo)
BORDER (borde)
PADDING (relleno interno)
CONTENT
(contenido)
┌─────────────────── MARGIN ────────────────────┐ │ │ │ ┌──────────────── BORDER ──────────────────┐ │ │ │ │ │ │ │ ┌───────────── PADDING ──────────────┐ │ │ │ │ │ │ │ │ │ │ │ ┌────────── CONTENT ───────────┐ │ │ │ │ │ │ │ │ │ │ │ │ │ │ │ Texto o elementos hijos │ │ │ │ │ │ │ │ │ │ │ │ │ │ │ └──────────────────────────────┘ │ │ │ │ │ │ │ │ │ │ │ └─────────────────────────────────────┘ │ │ │ │ │ │ │ └───────────────────────────────────────────┘ │ │ │ └────────────────────────────────────────────────┘
  1. Content (Contenido): El texto o elementos hijos
  2. Padding (Relleno): Espacio transparente entre el contenido y el borde
  3. Border (Borde): Línea que rodea el padding y el contenido
  4. Margin (Margen): Espacio transparente fuera del borde (separa de otros elementos)

1️⃣ Content (Contenido)

Es el área donde se muestra el contenido real: texto, imágenes, etc.

Área de contenido
width: 300px
height: 100px 
.caja {
    width: 300px;   /* Ancho del contenido */
    height: 100px;  /* Alto del contenido */
}
💡 Importante: Por defecto, width y height solo afectan al contenido, NO incluyen padding ni border.

2️⃣ Padding (Relleno Interno)

Espacio transparente entre el contenido y el borde. Usa el color de fondo del elemento.

Padding uniforme:

padding: 30px (igual en los 4 lados)
.caja {
    padding: 30px;  /* Aplica 30px a todos los lados */
}

Padding específico por lado:

padding-top: 10px
padding-right: 20px
padding-bottom: 30px
padding-left: 40px 
.caja {
    padding-top: 10px;
    padding-right: 20px;
    padding-bottom: 30px;
    padding-left: 40px;
    
    /* Shorthand equivalente: */
    padding: 10px 20px 30px 40px;  /* arriba, derecha, abajo, izquierda */
}

Atajos de padding:

Código Significado Ejemplo
padding: 20px; Todos los lados 20px en top, right, bottom, left
padding: 10px 20px; Vertical | Horizontal 10px arriba/abajo, 20px izquierda/derecha
padding: 10px 20px 30px; Top | H | Bottom 10px arriba, 20px lados, 30px abajo
padding: 10px 20px 30px 40px; ⏰ Sentido horario Arriba, Derecha, Abajo, Izquierda
📍 Truco mnemotécnico: Piensa en un reloj: empieza arriba (12) y sigue en sentido horario.

3️⃣ Border (Borde)

Línea visible que rodea el padding y el contenido.

Sintaxis del border:

border: [grosor] [estilo] [color];

/* Ejemplo: */
border: 2px solid #333;

Estilos de border:

solid
border: 4px solid #333;
dashed
border: 4px dashed #2196f3;
dotted
border: 4px dotted #4caf50;
double
border: 6px double #f44336;
groove
border: 6px groove #ff9800;
ridge
border: 6px ridge #9c27b0;
Bordes mixtos
Diferente estilo por lado
border-radius
border-radius: 20px;

Bordes individuales:

.caja {
    border-top: 2px solid red;
    border-right: 3px dashed blue;
    border-bottom: 4px dotted green;
    border-left: 5px double orange;
}

Border-radius (esquinas redondeadas):

border-radius: 10px;              /* Todas las esquinas */
border-radius: 10px 20px;         /* Superior-izq/inferior-der | Superior-der/inferior-izq */
border-radius: 10px 20px 30px 40px; /* Cada esquina (sentido horario desde arriba-izq) */

/* Esquinas individuales: */
border-top-left-radius: 10px;
border-top-right-radius: 20px;
border-bottom-right-radius: 30px;
border-bottom-left-radius: 40px;

/* Círculo perfecto: */
border-radius: 50%;

4️⃣ Margin (Margen Externo)

Espacio transparente FUERA del borde. Separa un elemento de otros elementos.

Margin para centrar:

margin: 30px auto;
(centrado horizontalmente) 
.centrado {
    width: 300px;
    margin: 30px auto;  /* auto centra horizontalmente */
}

Margin negativo:

Primera caja
Segunda caja con margin-top: -20px (se superpone) 
.superpuesta {
    margin-top: -20px;  /* Valores negativos permitidos */
}

Margin Collapse (colapso de márgenes):

⚠️ Comportamiento importante: Los márgenes verticales de elementos adyacentes se "fusionan" (toman el mayor). 
/* Estos dos elementos NO tendrán 60px de separación, sino 30px */
.elemento1 {
    margin-bottom: 30px;
}

.elemento2 {
    margin-top: 20px;
}

/* Resultado: 30px de separación (el mayor de los dos) */
Escenario Comportamiento
Márgenes verticales adyacentes Se colapsan (se usa el mayor)
Márgenes horizontales NO se colapsan (se suman)
Padre e hijo (primer/último hijo) Pueden colapsar si no hay padding/border
Elemento con padding o border NO colapsa

⚙️ Box-sizing: Cambiando el Comportamiento

La propiedad box-sizing controla cómo se calcula el tamaño total de una caja.

content-box (por defecto):

content-box
width: 200px
+ padding: 20px × 2
+ border: 10px × 2
= 260px total 
.content-box {
    box-sizing: content-box;  /* Valor por defecto */
    width: 200px;
    padding: 20px;
    border: 10px solid red;
}

/* Tamaño total renderizado: 260px de ancho */
/* 200 (content) + 40 (padding) + 20 (border) = 260px */

border-box (recomendado):

border-box
width: 200px
(incluye todo)
= 200px total 
.border-box {
    box-sizing: border-box;  /* ✅ Recomendado */
    width: 200px;
    padding: 20px;
    border: 10px solid green;
}

/* Tamaño total renderizado: 200px de ancho */
/* El padding y border se restan del width */
/* Content = 200 - 40 (padding) - 20 (border) = 140px */
🏆 Mejor práctica: Usar border-box globalmente: 
* {
    box-sizing: border-box;
}

/* Ahora width y height incluyen padding y border */
/* ¡Mucho más intuitivo para layouts! */
box-sizing width incluye Cálculo total
content-box Solo contenido width + padding + border
border-box ✅ Contenido + padding + border width (ya incluye todo)

📐 Calculando el Tamaño Total

Fórmula con content-box:

Ancho total = width + padding-left + padding-right + border-left + border-right + margin-left + margin-right

Alto total = height + padding-top + padding-bottom + border-top + border-bottom + margin-top + margin-bottom

Ejemplo práctico:

.caja {
    box-sizing: content-box;
    width: 300px;
    height: 200px;
    padding: 20px;
    border: 5px solid black;
    margin: 15px;
}

/* Cálculo del ancho total: */
300px (width)
+ 20px (padding-left)
+ 20px (padding-right)
+ 5px (border-left)
+ 5px (border-right)
+ 15px (margin-left)
+ 15px (margin-right)
─────────────────────
= 380px (ancho total en pantalla)

/* Cálculo del alto total: */
200px (height)
+ 20px (padding-top)
+ 20px (padding-bottom)
+ 5px (border-top)
+ 5px (border-bottom)
+ 15px (margin-top)
+ 15px (margin-bottom)
─────────────────────
= 280px (alto total en pantalla)
┌──────── 380px total (con margin) ─────────┐ │ │ │ ┌────── 350px (sin margin) ───────────┐ │ │ │ │ │ │ │ ┌────── 340px (sin border) ─────┐ │ │ │ │ │ │ │ │ │ │ │ ┌─── 300px (content) ─────┐ │ │ │ │ │ │ │ │ │ │ │ │ │ │ │ Contenido real │ │ │ │ │ │ │ │ │ │ │ │ │ │ │ └─────────────────────────┘ │ │ │ │ │ │ padding: 20px │ │ │ │ │ └───────────────────────────────┘ │ │ │ │ border: 5px │ │ │ └──────────────────────────────────────┘ │ │ margin: 15px │ └────────────────────────────────────────────┘

✅ Mejores Prácticas del Modelo de Caja

1. Usa border-box globalmente:

*, *::before, *::after {
    box-sizing: border-box;
}

2. Usa unidades consistentes:

/* ✅ Bueno */
.caja {
    padding: 1rem;
    margin: 2rem;
    border: 2px solid #333;
}

/* ❌ Malo (mezclar unidades sin razón) */
.caja {
    padding: 16px;
    margin: 1em;
    border: 0.125rem solid #333;
}

3. Prefiere padding sobre height para contenedores:

/* ✅ Bueno - padding permite que el contenido crezca */
.tarjeta {
    padding: 2rem;
}

/* ❌ Malo - height fijo puede cortar contenido */
.tarjeta {
    height: 200px;
    padding: 1rem;
}

4. Usa margin: 0 auto para centrar:

.contenedor {
    max-width: 1200px;
    margin: 0 auto;  /* Centra horizontalmente */
    padding: 0 1rem; /* Espaciado en móviles */
}

5. Ten cuidado con margin collapse:

/* Para evitar margin collapse, usa padding en el padre */
.padre {
    padding: 1px;  /* Previene margin collapse */
}

/* O usa flexbox/grid que no colapsan márgenes */
.padre {
    display: flex;
    flex-direction: column;
    gap: 1rem;  /* ✅ Mejor que margins */
}

6. Usa outline en lugar de border para debug:

/* ✅ outline no afecta el tamaño ni el layout */
* {
    outline: 1px solid red;
}

/* ❌ border sí afecta el tamaño */
* {
    border: 1px solid red;  /* Cambia dimensiones */
}

7. Aprovecha las propiedades lógicas (CSS moderno):

/* ✅ Propiedades lógicas (responsive a idiomas RTL/LTR) */
.caja {
    padding-block: 1rem;   /* top y bottom */
    padding-inline: 2rem;  /* left y right */
    margin-block-start: 2rem;  /* margin-top */
}

/* Tradicional */
.caja {
    padding-top: 1rem;
    padding-bottom: 1rem;
    padding-left: 2rem;
    padding-right: 2rem;
    margin-top: 2rem;
}

📋 Resumen Visual

Propiedad Transparente Visible Afecta tamaño Acepta negativos
margin ✅ Sí ❌ No ✅ Sí (espacio externo) ✅ Sí
border ❌ No ✅ Sí ✅ Sí ❌ No
padding ✅ Sí ❌ No (usa color de fondo) ✅ Sí ❌ No
content - ✅ Sí ✅ Sí -

💡 Recuerda:

  • Padding = espacio DENTRO del elemento (entre contenido y borde)
  • Margin = espacio FUERA del elemento (entre elementos)
  • Border = línea visible que separa padding y margin
  • box-sizing: border-box = simplifica cálculos (¡úsalo siempre!)
  • Margin collapse = márgenes verticales se fusionan
