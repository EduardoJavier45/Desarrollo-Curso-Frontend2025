🎨 Selectores CSS

Guía práctica con ejemplos interactivos

1. Selector de Elemento (Type Selector)

Selecciona todos los elementos de un tipo específico.

Código CSS:

p {
    color: #333;
    margin-bottom: 15px;
}

Resultado:

Este es un párrafo normal con el estilo aplicado.

Este es otro párrafo con el mismo estilo.

2. Selector de Clase (Class Selector)

Selecciona elementos con una clase específica. Prefijo: .

Código CSS:

.destacado {
    background-color: #fff3cd;
    border-left: 4px solid #ffc107;
    padding: 15px;
}

.importante {
    color: #dc3545;
    font-weight: bold;
}

Resultados:

Este div tiene la clase "destacado"

Este párrafo tiene la clase "importante"

Este párrafo tiene la clase "exito"

Este párrafo tiene la clase "info"

Múltiples clases:

Caja con dos clases: "caja" y "caja-azul"
Caja con dos clases: "caja" y "caja-verde"
Caja con dos clases: "caja" y "caja-roja"

3. Selector de ID

Selecciona un elemento único con un ID específico. Prefijo: #

Código CSS:

#encabezado-principal {
    font-size: 2.5rem;
    color: #667eea;
    text-align: center;
}

Resultado:

Este es el encabezado principal

⚠️ Recuerda: Los IDs deben ser únicos en la página

5. Selectores Descendientes

Selecciona elementos dentro de otros elementos.

Código CSS:

.destacado p {
    color: #856404;
}

Resultado:

Este párrafo está dentro de un div con clase "destacado"

Este también heredará el estilo

6. Selector Hijo Directo (>)

Selecciona solo los hijos directos.

Código CSS:

.lista-padre > li {
    background-color: #e3f2fd;
    padding: 10px;
}

Resultado:

  • Hijo directo (con estilo)
  • Hijo directo (con estilo)
  • Hijo directo (con estilo)
    • Hijo indirecto (sin estilo)
    • Hijo indirecto (sin estilo)

7. Selector Hermano Adyacente (+)

Selecciona el siguiente hermano inmediato.

Código CSS:

h3 + p {
    font-weight: bold;
    color: #667eea;
}

Ejemplo:

Este párrafo será DIFERENTE (en negrita y azul) porque viene justo después de h3.

Este párrafo será normal.

9. Pseudo-clases

Seleccionan elementos en estados específicos.

:hover (pasa el mouse sobre los botones)

Enlace como botón

:first-child y :last-child

  • Primer elemento (verde y negrita)
  • Elemento del medio
  • Elemento del medio
  • Último elemento (rojo y negrita)

:nth-child(odd) y :nth-child(even)

Fila 1 (impar - gris claro)
Fila 2 (par - gris medio)
Fila 3 (impar - gris claro)
Fila 4 (par - gris medio)

:focus (haz clic en los inputs)

10. Pseudo-elementos

Permiten estilizar partes específicas de elementos.

::before y ::after

Este texto tiene iconos antes y después

::first-letter

Esta es una letra capital decorativa al inicio del párrafo. El resto del texto sigue normalmente con su estilo predeterminado.

::first-line

La primera línea de este párrafo está en negrita y con color diferente. El resto del párrafo continúa con el estilo normal del texto.

11. Agrupación de Selectores

Aplicar los mismos estilos a múltiples selectores.

Código CSS:

h1, h2, h3, h4 {
    font-family: Georgia, serif;
}

Ejemplos de alertas:

Mensaje informativo
Mensaje de advertencia
Mensaje de error

12. Combinaciones Complejas

Puedes combinar múltiples selectores para mayor precisión.

Ejemplos de combinaciones:

  • div.destacado p - Párrafos dentro de divs con clase "destacado"
  • a.boton:hover - Enlaces con clase "boton" al pasar el mouse
  • input[type="text"]:focus - Inputs de texto cuando están enfocados
  • ul > li:first-child - Primer hijo directo de una lista
