# Desarrollo Curso Frontend 2025

Este repositorio contiene el material de estudio, ejercicios prácticos y proyectos desarrollados durante el bootcamp de **Desarrollo Front-End Trainee**. El contenido está organizado por módulos y sigue la estructura oficial del plan formativo aprobado por SENCE Chile.

**Programa:** Talento Digital para Chile

---

## 📚 Estructura del Curso - 9 Módulos

### Módulo 1: Orientación al Perfil y Metodología del Curso (18 horas)

**Contenido:**
- Introducción a la industria TI y el perfil de desarrollador Front-End
- Competencias técnicas y habilidades personales valoradas
- Metodología bootcamp y herramientas de trabajo colaborativo
- Importancia del portafolio de producto
- Trabajo en equipo, autoaprendizaje y comportamiento ético

---

### Módulo 2: Fundamentos de Desarrollo Front-End (72 horas)

**Contenido:**
- Introducción al desarrollo web (Front-End, Back-End, Fullstack)
- **HTML5:** Estructura de documentos, etiquetas semánticas, formularios
- **CSS:** Hojas de estilo, modelo de cajas, responsividad, Mobile First, Media Queries
- **Bootstrap:** Framework CSS para layouts y componentes
- **JavaScript básico:** Variables, condicionales, funciones, DOM
- **jQuery:** Manipulación del DOM y eventos
- **Git y GitHub:** Control de versiones, repositorios, colaboración

---

### Módulo 3: Desarrollo de la Interfaz de Usuario Web (56 horas)

**Contenido:**
- Del diseño a la implementación: rol UX/UI vs Front-End
- Metodologías de organización CSS: BEM, OOCSS, SMACCS
- **Preprocesador SASS:** Variables, mixins, parciales, patrón 7-1
- Modelo de cajas y Layout: Posicionamiento, tipos de layout
- **Bootstrap 4 avanzado:** Grillas, componentes JavaScript, extensiones con Sass

---

### Módulo 4: Fundamentos de Programación en JavaScript (60 horas)

**Contenido:**
- Características del lenguaje JavaScript
- Variables (simples y complejas), scope, operadores
- Estructuras de control: Condicionales (if/else)
- **Arreglos y ciclos:** Arrays, iteración (while, for), ciclos anidados
- **Funciones:** Definición, parámetros, retorno, alcance
- Objetos preconstruidos: Math, String
- Estilos y convenciones de codificación (código limpio)

---

### Módulo 5: Programación Avanzada en JavaScript (48 horas)

**Contenido:**
- **Programación Orientada a Objetos (POO):** Clases, objetos, herencia
- **Notación JSON:** Estructura y manipulación de datos
- **JavaScript ES6+:**
  - Variables: `let`, `const`
  - Arrow functions
  - Destructuring, spread/rest operators
  - Clases y módulos
  - Promises, async/await
- **DOM:** Manipulación y eventos
- **Programación asíncrona:** Callbacks, Promises, Async/Await
- **Consumo de APIs:** XMLHttpRequest (XHR), Fetch API
- APIs de terceros y procesamiento de respuestas JSON

---

### Módulo 6: Desarrollo de Interfaces Interactivas con Framework Vue (72 horas)

**Contenido:**
- **Introducción a Vue.js:** Framework progresivo, componentes web, SPA
- **Instalación:** CDN, Vue CLI, estructura de proyecto
- **Componentes Vue:** Morfología (template, script, style)
- **Templates y rendering:** Interpolaciones, directivas (`v-if`, `v-show`, `v-for`)
- **Form binding:** Two-way binding, `v-model`
- **Manejo de eventos:** onClick, modificadores de eventos
- **Vue Router:** Navegación, rutas estáticas/dinámicas/anidadas

---

### Módulo 7: Desarrollo de Aplicaciones Front-End con Framework Vue (76 horas)

**Contenido:**
- **Componentización avanzada:**
  - Comunicación padre-hijo (props, emit)
  - Ciclo de vida de componentes (hooks)
  - Slots y componentes dinámicos
- **Consumo de APIs REST:**
  - Conceptos de API REST, verbos HTTP (GET, POST, PUT, DELETE)
  - Librería Axios para peticiones HTTP
  - Firebase: Backend as a Service, autenticación, base de datos
- **Vuex:** Gestión de estado centralizado
  - State, Mutations, Actions, Getters, Modules
- **Testing:**
  - Pruebas unitarias: Jest, Mocha + Chai, Vue Test Utils
  - Pruebas end-to-end: Cypress, Nightwatch

---

### Módulo 8: Desarrollo de Portafolio de un Producto Digital (18 horas)

**Contenido:**
- Importancia del portafolio profesional
- Buenas prácticas para crear un portafolio
- **Herramientas:**
  - GitHub para desarrolladores
  - Servicios de hosting gratuitos
  - YouTube para tutoriales técnicos
- Finalización y presentación del proyecto integrador

---

### Módulo 9: Desarrollo de Empleabilidad en la Industria Digital (18 horas)

**Contenido:**
- **Mercado laboral TI:** Características, procesos de selección
- **Plan de búsqueda laboral:**
  - Elaboración de CV técnico
  - LinkedIn y portales laborales (GetOnBoard, Trabajando, Laborum)
- **Preparación de entrevistas:**
  - Elevator pitch
  - Desafíos técnicos
  - Tests psicolaborales
  - Gestión emocional y escucha activa
- Competencias técnicas vs habilidades blandas valoradas

## 🚀 Cómo Usar Este Repositorio

1. **Clonar el repositorio:**
   ```bash
   git clone https://github.com/EduardoJavier45/Desarrollo-Curso-Frontend2025.git
   cd Desarrollo-Curso-Frontend2025
   ```

2. **Navegar por módulos:**
   Cada carpeta contiene ejercicios, proyectos y recursos específicos del módulo.

3. **Instalar dependencias (cuando sea necesario):**
   ```bash
   npm install
   ```

4. **Ejecutar proyectos:**
   ```bash
   npm run dev
   # o
   npm run serve
   ```
